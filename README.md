# NeuroCraft — курс по нейросетям для дизайнеров

Одностраничный лендинг курса «Нейросети для коммерческих дизайнеров и создателей контента».

## Стек

- Vite + React + TypeScript
- Tailwind CSS v4
- Framer Motion

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```
